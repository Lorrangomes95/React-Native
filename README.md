# React Native Project



## 📋 Sobre o Projeto

Este é um aplicativo desenvolvido em **React Native**, criado como parte do evento **NLW Pocket: Mobile - React Native** da **Rocketseat**. O objetivo do projeto é [descreva brevemente o propósito ou a funcionalidade principal do projeto].

O desenvolvimento incluiu conceitos como:
- Propriedades, Estados e Componentes.

- Roteamento com Expo Router.
- Integração com APIs.
- Leitura de QRCode e exibição de mapas.

## 🛠️ Como Executar o Projeto

1. Clone este repositório:

```bash
git clone https://github.com/Lorrangomes95/React-Native.git
```

2. Navegue até o diretório do projeto:

```bash
cd React-Native
```

3. Instale as dependências:

```bash
npm install
# ou
yarn install
```

4. Execute o projeto:

```bash
npm start
# ou
expo start (se estiver usando Expo)
```

## 💡 Funcionalidades

- Leitura de QRCode utilizando a câmera do dispositivo.
- Exibição de mapas integrados com APIs externas.
- Navegação entre telas utilizando roteamento.
- Consumo de API Node.js.
